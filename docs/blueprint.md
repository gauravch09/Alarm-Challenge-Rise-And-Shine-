# **App Name**: Rise & Shine Challenge

## Core Features:

- Alarm Setting: Users can set and manage multiple alarms with specific times and recurrence patterns.
- Step Challenge: To stop the alarm, users must walk a predetermined number of steps, tracked via the phone's sensors.
- Puzzle Challenge: Users must solve a dynamically generated puzzle (e.g., unique math problem or memory sequence tool) to dismiss the alarm.
- Photo Challenge: The alarm is dismissed only after the user takes a photo that matches a pre-set image (e.g., bathroom sink).
- No Easy Exit Mode: When an alarm is active, the app prevents snoozing and makes it difficult to exit, ensuring task completion.

## Style Guidelines:

- Color scheme: A bright and clear light theme for an energetic start to the day. Primary color: A vibrant yet refreshing blue (#267AFF) to convey clarity and freshness. Background color: A very light, subtle blue (#EAF2F7) to maintain a clean and airy feel. Accent color: A dynamic turquoise (#19E3E3) for highlighting important actions and indicators.
- Body and headline font: 'Inter' (sans-serif) for its modern, clean, and highly readable qualities, suitable for instructions and clear task display.
- Utilize simple, clear line-art icons that are intuitive and directly represent the challenges (e.g., footprints for steps, puzzle pieces, camera icon).
- The layout should be clean and uncluttered, with primary focus given to the active alarm and the challenge description. Key action buttons should be prominent and easy to identify, guiding the user through the challenge.
- Implement subtle, smooth animations for transitions between screens, progress indication during challenges (e.g., step counter, puzzle interaction), and celebratory feedback upon alarm dismissal.
